# Figure 5

This folder contains the necessary data to reproduce Figure 5, showing the evidence between genes *B3GNT5* and *ST6GAL1*. 
`PrepPlotGainLoss.R` is a helper file included in the main file, `Fig4PrepPlot.R`. The datafile loaded contains cleaned information from the other analyses to make plotting easier.
This analysis also requires `ModulePredsAllPairs.RData`, which is available from Zenodo.
